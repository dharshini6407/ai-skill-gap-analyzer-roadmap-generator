# AI-Powered Skill Gap Analyzer and Learning Roadmap Generator

## Project Title

**AI-Powered Skill Gap Analyzer and Learning Roadmap Generator**

---

## Problem Statement

Many students and job seekers struggle to identify the skills required for their desired careers. They often lack awareness of the gap between their current skill set and industry expectations. Existing career guidance methods are mostly generic and do not provide personalized learning paths based on individual profiles and career goals.

This project aims to develop an AI-powered system that analyzes a user's skills, compares them with job role requirements, identifies skill gaps, and generates a personalized learning roadmap to improve career readiness.

---

## Project Objectives

1. To allow users to create accounts and manage their profiles.
2. To upload and analyze resumes for skill extraction.
3. To identify existing technical and professional skills.
4. To compare user skills with selected job role requirements.
5. To perform skill gap analysis and highlight missing skills.
6. To calculate a Career Readiness Score.
7. To generate personalized learning path recommendations.
8. To create AI-powered learning roadmaps for career growth.
9. To track user progress throughout the learning journey.
10. To provide a dashboard for monitoring career development.
